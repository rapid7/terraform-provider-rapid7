# Rapid7 Terraform Provider

The [Rapid7 Terraform provider](https://registry.terraform.io/providers/rapid7/rapid7)
provides the ability to manage [Command Platform API](https://docs.rapid7.com/insight/api-overview/)
resources via Terraform.

Please refer to [Rapid7 Documentation](https://docs.rapid7.com/insightidr/detections-as-code) and the full provider documentation [on the Terraform registry](https://registry.terraform.io/providers/rapid7/rapid7/latest/docs) for further information.

## Requirements

The Rapid7 provider requires at least [Terraform CLI 1.0](https://developer.hashicorp.com/terraform/install)
