# variable (function)

Generates a LEQL variable interpolation for referencing a variable resource managed by Terraform.
Using this function establishes a dependency link between the variable and rule in Terraform, ensuring that any variables are created before rules referencing them.



## Example

```terraform
resource "rapid7_siem_variable" "example_rule_variable" {
  name  = "rule_variable"
  value = "some, values"
}

resource "rapid7_siem_detection_rule" "example_rule_with_variable" {
  name        = "Test"
  description = "Test"

  action   = "CREATES_ALERTS"
  priority = "LOW"

  logic = <<-LEQL
    from(
      event_type = process_start_event
    )
    where(
      this = ${provider::rapid7::variable(rapid7_siem_variable.example_rule_variable)}
    )
  LEQL
}
```
