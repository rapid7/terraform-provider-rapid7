# rapid7_siem_log_search_query (Data Source)

Use this data source to retrieve the results of a [log search query](https://docs.rapid7.com/insightidr/log-search/#search-your-log-data)



## Example

```terraform
data "rapid7_siem_log_search_query" "example_query" {
  logs = ["e57ac71c-2506-4fcc-8dd1-83628eee4140"]

  query = "where(os_type = WINDOWS)"

  range = {
    # Last 2 hours
    from = timeadd(plantimestamp(), "-2h")
    to   = plantimestamp()
  }

  # Optional
  order = "DESC"
  limit = 10
}
```
