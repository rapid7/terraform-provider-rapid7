# Getting Started

## Terraform Basics

It is recommended that first you have a base understanding of what Terraform is and how it operates.

- If you are new to Terraform, please familiarize yourself by reading the [Introduction to Terraform](https://developer.hashicorp.com/terraform/intro) and [Core Workflow](https://developer.hashicorp.com/terraform/intro/core-workflow) documentation.

- If you have never used Terraform, please refer to the [installation instructions](https://developer.hashicorp.com/terraform/install) to install Terraform in your development environment.

- You should also familiarize yourself with the Terraform [configuration language and syntax](https://developer.hashicorp.com/terraform/language) and [files and configuration structure](https://developer.hashicorp.com/terraform/language/files)

## The Rapid7 Terraform Provider

The Rapid7 Terraform provider is a convenient wrapper around [the Command Platform APIs](https://docs.rapid7.com/insight/api-overview), enabling Terraform management of resources relating to Command Platform products and features.

[Learn more about Terraform providers](https://developer.hashicorp.com/terraform/language/providers)

When you are ready to begin using the Rapid7 Terraform provider:
* Understand how to [specify required providers](https://developer.hashicorp.com/terraform/language/providers/requirements)
* Copy the code snippet from the "Use Provider" link at the top right of this documentation page
* This specifies a requirement for the version of the Rapid7 provider that this documentation is for
